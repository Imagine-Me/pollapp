# Shell
Shell hold together all the other microfrontends.

## :hammer: Development
### Creating .env files
Create a `.env` file in folder `shell`. Copy everything from `.env.sample` and refactor if required
### Installing packages
```sh
yarn install
```
### Development
```sh
yarn dev
```
> `shell` is depend on `authentication`,`poll` and `profile`
### Building
```sh
yarn build
```
