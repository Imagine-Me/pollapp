# Utils
Utils serve some of the common utils such as axios instance, socket etc...
## :hammer: Development

### Creating .env files
Create a `.env` file in folder `utils`. Copy everything from `.env.sample` and refactor if required
### Installing packages
```sh
yarn install
```
### Development
```sh
yarn dev
```
> `utils` is depend on `authentication`
### Building
```sh
yarn build
```
